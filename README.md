# AI Creative Pass

Premium digital gift card experience for AI subscriptions.

## Features

- 3D flip card with touch/swipe interaction
- Landing unboxing animation
- Mobile-first responsive design
- Zero dependencies (vanilla HTML/CSS/JS)

## Usage

1. Scan QR code with mobile device
2. Watch the unboxing animation
3. View gift card details (front)
4. Flip to see usage instructions (back)
